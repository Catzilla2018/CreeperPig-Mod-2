# 🐷💣 Creeper Pig Mod

A Minecraft Forge mod for version 1.7.10 that replaces Pig behavior with Creeper behavior!

## Features
- Pigs are now hostile and attack players
- They explode when close to targets
- Fuse timer before detonation
- Block damage enabled

## Installation
1. Download the latest JAR from Releases
2. Place in your `.minecraft/mods/` folder
3. Launch Minecraft with Forge 1.7.10

## Building
```bash
./gradlew build
```
JAR will be at `build/libs/CreeperPig-1.0.0.jar`
